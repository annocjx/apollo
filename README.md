# Apollo

#### 项目介绍
apollo项目是一个使用springboot开发微服务项目。本人闲来无事想开发一个开源社区，用于移动端开发人员查阅相关的开源框架，以提高开发效率而开发的此项目。
该项目使用IntelliJ IDE，基于springboot2.1.0正式版、jdk8构建。

#### 软件架构
项目技术栈：SpringBoot + MyBatis + Mysql + redis + swagger2


#### 项目环境

1. 安装jdk环境
2. 安装maven，配置maven环境
3. Intellij ide
4. mysql 5.7
5. redis
